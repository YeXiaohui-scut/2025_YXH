# Latent-WOFA Project\n\nThis project implements the Latent-WOFA framework for advanced noise compilation and embedding.\n\n## Requirements\n- Python 3.8+\n- PyTorch 1.9+\n- torchvision\n- other dependencies listed in `requirements.txt`\n\n## Project Structure\n\n```plaintext\nlatent_wofa/\n├── README.md\n├── requirements.txt\n├── configs/\n│   └── config.yaml\n├── models/\n│   ├── __init__.py\n│   ├── stage1_codec.py\n│   ├── stage2_embedder.py\n│   ├── stage2_extractor.py\n│   └── distortion_layers.py\n├── utils/\n│   ├── __init__.py\n│   ├── progressive_curriculum.py\n│   └── metrics.py\n├── train_stage1.py\n├── train_stage2.py\n├── inference.py\n└── eval.py\n```\n\n## Overview of Stages\n\n1. **Stage I**: Train a noise encoder/decoder in pixel space.\n2. **Stage II**: Train an embedder and extractor in latent space using a pre-trained VAE from Stable Diffusion.\n3. **Progressive Distortion Layers**: Implement various attacks to evaluate the robustness of the model.\n4. **Curriculum Learning**: Apply progressive curriculum for enhanced training efficiency.\n\n## Usage\nInclude instructions on how to run the training scripts, inference, and evaluations.
